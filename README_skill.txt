技の種類
■damage
 ・normal           固定ダメージ
 ・coin
   ▸finite          コインの枚数(有限)×
　 ▸infinite        コインの枚数(無限)×
 ・side             サイドカードの枚数×
 ・energy           エネルギーカードの枚数×
 ・dmg_count        
   ▸opponent        相手のバトルポケモンにダメカンが載っているなら追加ダメージ
   ▸self            技を出すポケモンに乗っているダメカン× (追加ダメージが負の場合あり)
   ▸bench           自分のベンチポケモンにダメカンが載っているなら追加ダメージ
 ・trush      
   ▸opponent        相手のトラッシュにある基本エネルギー(サポート)の枚数×
   ▸self　　　      自分のトラッシュにある基本エネルギー(サポート)の枚数×
   ▸add             捨てた枚数×追加ダメージ
 ・assignment       ダメカンx個を相手のポケモンに好きなように載せる(ダメージを自由に配分)
 ・bench      
   ▸opponent        相手のベンチポケモンにダメージを与える
   ▸self            自分のベンチポケモンにダメージを与える
 ・continuous       連続で技を繰り出す
 ・backlash         前のターンに自分のポケモンが気絶していたなら追加ダメージ
 ・characteristic   相手の場の特性を持つポケモンの数×
 ・tag              自分のベンチの同じ技を持つポケモンの数×
 ・self destraction 
   ▸coin            コインが表なら自分にもダメージ
   ▸constant        自分に一律ダメージ
 ・evolution        相手のバトルポケモンが進化ポケモンなら追加ダメージ

■status
 ・毒
 ・まひ
 ・眠り
 ・氷
 ・やけど
 ・こんらん

■heal         
 ・normal           技を出すポケモンの体力を回復
 ・all bench        自分のベンチポケモン全員の体力を回復


■reaction          次のターンに技が使用不可

■restraint         この技を受けたポケモンは次のターン逃げられない

■decrease     
 ・opponent         この技を受けたポケモンは次のターン使う技のダメージが減少 
 ・self

■trash
 ・energy      
   ▸opponent        相手のポケモンについているエネルギーカードを捨てる
   ▸self            このポケモンについているエネルギーカードを捨てる
 ・hand             相手の手札を見ないで捨てる

■defense           
 ・coin             コインが表なら次のターン技のダメージや効果を受けない
 ・constant         次の相手の番このポケモンが受ける技のダメージを減少

■draw
 ・above            山札の上から引く
 ・free             山札から自由に引く
 ・hand             手札の合計がx枚になるように引く
 ・constant         x枚引く

■back              自分のトラッシュの基本エネルギーを山札に戻してシャッフル

■trap              次の相手の番, このポケモンが技のダメージを受けたとき, 技を使ったポケモンにダメージ

■change
 ・monster
   ▸opponent         相手はバトルポケモンとベンチポケモンを入れ替える
   ▸self             自分はバトルポケモンをベンチポケモンを入れ替える
 ・energy            このポケモンについているエネルギーをベンチポケモンに好きなように付け替える

■invalid            
 ・resistance        この技のダメージは抵抗力を計算しない
 ・effect            この技のダメージは相手のバトルポケモンにかかっている効果を計算しない

■degeneration       相手の進化しているバトルポケモンから進化カードを1枚はがす はがしたカードは手札へ
